# 🎧 About Project

**Krishan Music** is a modern web-based music platform designed to deliver a smooth and beautiful music listening experience.

This project focuses on performance, design, and smooth user interaction.

The entire system is **conceptualized, designed, and developed by Krishan**.

---

# ⚡ Features

- 🎵 Music Player Interface  
- 🎧 Playlist Support  
- 🌙 Dark Mode UI  
- ⚡ Fast Loading  
- 📱 Mobile Responsive  
- 🎨 Smooth Animations  

---

# 🧠 Tech Stack

| Technology | Purpose |
|------------|--------|
| HTML5 | Structure |
| CSS3 | UI Design |
| JavaScript | Logic |
| Web Audio API | Music playback |
